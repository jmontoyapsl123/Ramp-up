Docker scripts:
- sudo docker build . -t <name-image> => create image
- sudo docker ps => list run container
- sudo docker images => list created images
- sudo docker run --rm --net <name-network> --name <name-container> => Run images and put it in a network
- sudo docker network create <name-network> => create a network
- sudo docker network ls => List all networks 
- sudo docker rmi <image-id> -f => force to delete a docker images
- sudo docker rm <container-id> -f => force to delete a docker container
